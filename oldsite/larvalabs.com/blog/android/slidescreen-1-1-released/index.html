SlideScreen 1.1 Released

by Matt Hall on Android /

After a bit of an extended testing period, SlideScreen 1.1 has been released to the market. This is a pretty big release, so here’s the change list:

  • Facebook! You can now load your Facebook news stream and view entries in the built in reader. Thanks to FBConnect Android (http://code.google.com/p/fbconnect-android/) for the Faceook login support.
  • Google Reader upgrades:
    • Move between items within the reader. Just swipe the title, or use the volume buttons.
    • Automatically mark items as read when viewing them.
    • Sync only some labels/folders from your subscriptions.
    • Added feed name to title bar.
    • Specify account to load subscriptions from, overrides the built in account on your phone.
    • Character set fixes.
  • Twitter upgrades:
    • Load mentions into your timeline.
    • Better profile image caching for faster loading/browsing.
    • Swipe title or use volume buttons to navigate between tweets in the reader.
  • Stocks upgrade:
    • Specify account to load portfolio from, overrides the built in account on your phone.
  • GPS based weather location. (This is still considered in beta, but it seems to work pretty well).
  • Bug fix: Calendar entries showing up early/late based on timezone error.
  • Bug fix: Calendar names too long for screen.
  • Bug fix: International symbols in stock feeds causing incorrect values.
  • Bug fix: Battery level inconsistent with system level.
  • Bug fix: International characters in news feeds being displayed incorrectly (also fixed for tweets).
  • Bug fix: When in the GMail app, unable to see All Mail.
  • Bug fix: Google Reader update could get stuck in some situations.
  • Bug fix: Crash when long pressing on SMS icon on an HTC Sense phone.
  • Bug fix: Search button did nothing on HTC Sense phones.
  • Bug fix: Orientation change (keyboard open or device tilt) could cause settings and setup wizard to restart.

Hopefully we didn’t forget anything. Give it a try and tell us what you think! Don’t forget we try to work on features in the order of voting on the support site so go there to vote for your favorites.

Thanks for all the great feedback, hope you like it!
